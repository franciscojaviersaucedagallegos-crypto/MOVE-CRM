# 🔥 MOVE CRM — Ingeniería Contra Incendios

CRM comercial para el equipo de ventas de MOVE Ingeniería Contra Incendios.

## 📱 Instalar en iPhone

1. Abre el link de arriba en **Safari**
2. Toca el botón **Compartir ↑**
3. Selecciona **"Añadir a pantalla de inicio"**
4. Toca **Agregar**

## 💻 Instalar en Mac

1. Abre el link en **Chrome**
2. Menú **⋮ → Instalar MOVE CRM...**

## ✨ Funciones

- **📊 Seguimiento** — Ciclo de 30 días por prospecto con semáforo de prioridad
- **👥 Contactos** — ID automático por cliente/prospecto, bitácora de actividad
- **📧 Campañas** — Email individual editable o BCC masivo con adjuntos de Drive
- **📋 Cotizaciones** — Numeración automática COT-YYYY-001, PDF, registro de factura
- **📈 KPI** — Conversión comercial y financiera, pipeline, tendencias mensuales
- **☁️ Sincronización** — Datos compartidos en tiempo real vía Google Sheets

## 👥 Usuarios

App diseñada para 3 usuarias del equipo comercial MOVE.  
Los datos se sincronizan automáticamente entre todos los dispositivos.

## 🛠 Tecnología

- HTML + JavaScript puro (sin frameworks)
- Google Sheets como base de datos compartida
- Google Apps Script como API
- GitHub Pages como servidor
